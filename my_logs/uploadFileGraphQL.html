code hướng dẫn truyền file bằng graphQL

Để truyền file bằng GraphQL, bạn cần sử dụng kiểu dữ liệu Upload được hỗ trợ bởi các thư viện GraphQL. Trong hướng dẫn này, tôi sẽ sử dụng Node.js và thư viện Apollo GraphQL để minh họa việc truyền file.

Đầu tiên, bạn cần cài đặt Apollo Server và các thư viện cần thiết để xử lý file. Bạn có thể cài đặt chúng bằng cách chạy các lệnh sau trong terminal:

npm install apollo-server graphql apollo-upload-server multer graphql-upload

Tiếp theo, tạo một file server.js với nội dung sau:

javascript
const { ApolloServer, gql } = require("apollo-server");
const { createReadStream } = require("fs");
const { resolve } = require("path");
const { ApolloServerPluginLandingPageGraphQLPlayground } = require("apollo-server-core");
const { ApolloServerPluginLandingPageDisabled } = require("apollo-server-core");
const { graphqlUploadExpress } = require("graphql-upload");
const multer = require("multer");

const typeDefs = gql`
  scalar Upload

  type Query {
    hello: String
  }

  type Mutation {
    uploadFile(file: Upload!): String
  }
`;

const resolvers = {
  Query: {
    hello: () => "Hello world!",
  },

  Mutation: {
    uploadFile: async (parent, { file }) => {
      const { createWriteStream, unlink } = require("fs");
      const { v4: uuidv4 } = require("uuid");
      const { filename, mimetype, createReadStream } = await file;
      const stream = createReadStream();
      const id = uuidv4();
      const path = `./uploads/${id}-${filename}`;
      await new Promise((resolve) =>
        stream
          .pipe(createWriteStream(path))
          .on("finish", () => resolve())
          .on("error", (error) => {
            unlink(path, () => {});
            console.log(error);
          })
      );
      return `http://localhost:4000/${path}`;
    },
  },
};

const server = new ApolloServer({
  typeDefs,
  resolvers,
  plugins: [process.env.NODE_ENV === "production" ? ApolloServerPluginLandingPageDisabled() : ApolloServerPluginLandingPageGraphQLPlayground()],
});

const upload = multer();

server.start().then(() => {
  server.applyMiddleware({
    app: upload.none(),
    path: "/graphql",
  });
});

Trong file server.js này, chúng ta định nghĩa kiểu dữ liệu Upload là một scalar, định nghĩa một truy vấn đơn giản để kiểm tra server đã hoạt động hay chưa và định nghĩa một mutation để truyền file. Trong mutation này, chúng ta sử dụng thư viện multer để xử lý file và lưu file vào thư mục uploads. Sau đó, mutation sẽ trả về đường dẫn đến file vừa được lưu.

code hướng dẫn upload file bằng graphQL trong nestjs

Để upload file bằng GraphQL trong NestJS, bạn có thể sử dụng thư viện graphql-upload. Trong hướng dẫn này, tôi sẽ hướng dẫn cách upload file bằng GraphQL trong NestJS bằng cách sử dụng thư viện này.

Đầu tiên, bạn cần cài đặt thư viện graphql-upload bằng lệnh sau:

npm install graphql-upload

Tiếp theo, bạn cần định nghĩa graphqlUploadExpress middleware trong main.ts để sử dụng thư viện graphql-upload. Bạn có thể định nghĩa middleware như sau:

typescript
import { NestFactory } from '@nestjs/core';
import { AppModule } from './app.module';
import { graphqlUploadExpress } from 'graphql-upload';

async function bootstrap() {
  const app = await NestFactory.create(AppModule);
  app.use(graphqlUploadExpress({ maxFileSize: 10000000, maxFiles: 10 }));
  await app.listen(3000);
}
bootstrap();

Trong đoạn code trên, chúng ta sử dụng graphqlUploadExpress middleware với các tùy chọn để giới hạn kích thước file và số lượng file tối đa.

Tiếp theo, trong module của bạn, hãy import GraphQLUpload scalar từ graphql-upload và định nghĩa mutation để upload file. Ví dụ như sau:

typescript
import { Resolver, Mutation, Args, Query } from '@nestjs/graphql';
import { GraphQLUpload, FileUpload } from 'graphql-upload';

@Resolver()
export class AppResolver {
  @Mutation(() => String)
  async uploadFile(
    @Args({ name: 'file', type: () => GraphQLUpload })
    file: FileUpload,
  ): Promise<string> {
    const { createReadStream, filename } = file;
    const stream = createReadStream();
    // Xử lý stream tùy theo nhu cầu của bạn
    return `File ${filename} đã được upload thành công.`;
  }
}

Trong đoạn code trên, chúng ta định nghĩa mutation để upload file bằng cách sử dụng scalar GraphQLUpload và tham số file có kiểu FileUpload từ thư viện graphql-upload. Sau đó, chúng ta có thể xử lý stream của file và trả về kết quả tùy thuộc vào nhu cầu của bạn.

Bây giờ bạn có thể thực hiện các yêu cầu GraphQL để upload file bằng GraphQL trong NestJS. Ví dụ như sau:

graphql
mutation uploadFile($file: Upload!) {
  uploadFile(file: $file)
}

Lưu ý rằng, bạn cần sử dụng multipart/form-data để gửi file trong yêu cầu GraphQL. Bạn cũng có thể sử dụng thư viện apollo-upload-client để upload file bằng GraphQL trong các ứng dụng client.
